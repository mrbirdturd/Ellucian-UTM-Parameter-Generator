Ellucian UTM Parameter Builder

How to Use the UTM Parameter Builder Form

The Ellucian UTM Parameter Builder is designed to help marketers generate UTM URLs easily for their campaigns. Follow these steps to use the form:

  1. Base URL: Enter the base URL of your website (e.g., ellucian.com). The prefix https://www. will be automatically added to the generated URL.
  2. UTM Source: Select the source of your traffic from the dropdown list (e.g., Google, Facebook). If you do not want to include a UTM source, select "None" or leave it blank.
  3. UTM Medium: Choose the medium through which the traffic arrives (e.g., email, social). Similar to UTM Source, you can leave this blank or select "None" if you prefer not to include it.
  4. UTM Campaign: Enter the campaign name. This field will automatically convert spaces to hyphens and capitalize all letters to lowercase (e.g., "20240626 EU NTN AMOSSHE" becomes "20240626-eu-ntn-amosshe").
  5. UTM Campaign ID: If applicable, input the campaign ID (e.g., 00vNu000005KKuBIAW).
  6. UTM Content: Add additional content information if needed (e.g., elive24).
  7. Generate UTM URL: Click the "Generate UTM URL" button to create the URL. If you make any changes to the fields, simply click the button again to refresh the URL without needing to refresh the page.
  8. Copy UTM URL: Use the "Copy URL" button to copy the generated URL to your clipboard for easy use.
